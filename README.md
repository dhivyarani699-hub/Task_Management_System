Task Management Application

Description
A simple Task Management Application built using HTML, CSS, and JavaScript. Users can add, edit, and delete tasks. The application stores tasks using browser local storage.

Features

Add Tasks
Edit Tasks
Delete Tasks
Local Storage Support
Responsive Design

Technologies Used

HTML
CSS
JavaScript

How to Run
Open index.html in a web browser.
Add and manage your tasks.
